# Т-Библиотека

Консольное приложение для ведения личной библиотеки. Позволяет хранить список книг, отслеживать прочитанное и быстро находить нужную книгу. Все данные сохраняются локально в JSON-файл и не теряются между запусками.

Написано на чистом Python 3, без сторонних зависимостей.

---

## Возможности

- Добавление книги с указанием названия, автора, жанра, года издания и описания
- Просмотр всей библиотеки с сортировкой по названию, автору или году
- Фильтрация по жанру и статусу (прочитана / не прочитана)
- Полнотекстовый поиск по названию, автору и описанию
- Список избранного
- Отметка книг как прочитанных
- Удаление книг из библиотеки

---

## Запуск

```bash
python main.py
```

Никаких дополнительных установок не требуется. Файл `library.json` создаётся автоматически при первом добавлении книги.

---

## Структура проекта

```
├── main.py       # точка входа
├── cli.py        # консольный интерфейс
├── library.py    # основная логика
├── storage.py    # чтение и запись данных
└── models.py     # модель книги
```
